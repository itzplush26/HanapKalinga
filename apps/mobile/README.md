# HanapKalinga Mobile App

React Native mobile application for the HanapKalinga platform (Coming Soon).

## Planned Tech Stack

- React Native with Expo
- TypeScript
- Supabase for backend
- React Navigation for routing
- NativeWind or React Native Paper for UI

## Development

_Setup instructions will be added when mobile app development begins._

## Shared Code

This mobile app will share business logic with the web app through the `@hanapkalinga/shared` package:

- Constants
- Type definitions
- Validation schemas
- API client configurations
- Utility functions

The UI will be implemented independently using React Native components.
